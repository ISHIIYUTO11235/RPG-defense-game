# RPG-defense-game

JavaScript で実装した、**RPG要素のあるブラウザ防衛ゲーム**。押し寄せる敵を迎え撃ちながら、RPG的な成長・強化の要素を取り入れたゲームです。

## 概要

ブラウザ上で動作する2D防衛ゲーム。敵の進行を食い止める「ディフェンス」の面白さに、ユニットやステータスの強化といったRPG的な成長要素を組み合わせています。ゲームループ・当たり判定・敵の出現管理などをJavaScriptで実装しました。

## 主な要素

- **防衛（ディフェンス）**: ウェーブ状に出現する敵を迎撃
- **RPG要素**: 強化・成長によるプレイヤー側のパワーアップ
- リアルタイムな更新 → 描画のゲームループ

## 工夫した点

- 敵のウェーブ管理・当たり判定・スコア/状態管理など、**ゲームの基本システムを自前で設計・実装**。
- 「防衛ゲーム」に「RPGの成長」を掛け合わせ、単発の操作で終わらない**継続的にプレイしたくなるゲーム性**を意識した。
- ブラウザだけで完結し、インストール不要で誰でも遊べる構成。

## 使用技術

- **言語**: JavaScript
- **描画**: HTML5 Canvas（ブラウザ上で動作）

## 遊び方

リポジトリをクローン、または ZIP でダウンロードし、`index.html` をブラウザで開くとプレイできます。

> ゲーム開発（ゲームループ・状態管理・当たり判定）の実践を目的としたプロジェクトです。
